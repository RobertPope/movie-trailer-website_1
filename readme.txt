This Python program generates a webpage that show a selection of trailers

In order to run this program you must have the following installed on your machine - Python 2.7.9 or better

In order to add or change movies to the website you will need to edit the entertainment_center.py

Adding a movie:

Example

avatar = media.Movie("Avatar",
                     "A Marine on an alean planet",
                     "http://upload.wikimedia.org/wikipedia/en/b/b0/Avatar-Teaser-Poster.jpg",
                     "https://youtu.be/5PSNL1qE6VY")

use the media.Movie class to create a movie.  you can paste this code immediately
above the "movies" variable.  Make sure you include your new movie on the list

To launch the entertainment center, navigate to this directory in your terminal
execute the following command - 
python entertainment_center.py
